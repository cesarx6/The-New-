# Proyectos de análisis -

Analista de datos enfocado en transformar información en decisiones.

En este portafolio presento proyectos desarrollados bajo el marco OSEM, donde abordo problemas reales mediante procesos estructurados de limpieza, validación y análisis de datos.

Mi enfoque se centra en generar insights claros, medibles y accionables, comunicados a través de reportes y visualizaciones que facilitan la toma de decisiones estratégicas.
